### Statistics & Mathematics

#### Overview

A strong foundation in statistics and mathematics is essential for any data scientist. These disciplines provide the theoretical framework for understanding data, building predictive models, and interpreting results. Key areas include descriptive statistics (summarizing data with measures like mean and median), probability (quantifying uncertainty), and linear algebra (the basis for many machine learning algorithms). Mastering these concepts will help you understand the "why" behind the code and make you a more effective and insightful practitioner.

#### Projects/Exercises

1.  **Descriptive Statistics with Python**: Find a simple dataset (e.g., the Iris dataset) and use Python's `statistics` library to calculate the mean, median, mode, and standard deviation for each numerical column.
2.  **Explore Statistical Notebooks**: Clone the [Stats-Maths-with-Python](https://github.com/tirthajyoti/Stats-Maths-with-Python) repository. Run the notebooks on topics that interest you, such as probability distributions or hypothesis testing, and try to modify the code to see how the results change.
3.  **Explain a Concept**: After watching a video on a core concept like the Central Limit Theorem or p-values, try to write a simple explanation in your own words. Create a small Python script to generate data that demonstrates the concept.
