# Essential Books — DS, ML, AI Engineering

> Curated reading list with difficulty ratings, notes on who each book is for, and links to free versions where available.

**Difficulty:** Beginner | Intermediate | Advanced | Expert

---

## Data Science Books

| # | Title | Author | Level | Free? | Notes |
|---|-------|--------|-------|-------|-------|
| 1 | Python Data Science Handbook | Jake VanderPlas | Beginner-Int | [Yes (GitHub)](https://github.com/jakevdp/PythonDataScienceHandbook) | Best intro to numpy/pandas/matplotlib/sklearn |
| 2 | Python for Data Analysis | Wes McKinney | Beginner-Int | No (cheap) | The definitive pandas book — by the creator |
| 3 | Think Stats | Allen Downey | Beginner | [Yes (free)](https://greenteapress.com/thinkstats2/) | Statistics with Python, programmer-friendly |
| 4 | Think Bayes | Allen Downey | Intermediate | [Yes (free)](https://greenteapress.com/thinkbayes2/) | Bayesian statistics in Python — excellent |
| 5 | Naked Statistics | Charles Wheelan | Beginner | No | Non-technical, great for intuition |
| 6 | The Art of Statistics | David Spiegelhalter | Beginner-Int | No | Modern, readable statistics book |
| 7 | Data Science from Scratch | Joel Grus | Beginner-Int | No | Implements DS algorithms from scratch — great for understanding |
| 8 | Storytelling with Data | Cole Nussbaumer Knaflic | Beginner | No | Essential for data visualization communication |
| 9 | R for Data Science | Hadley Wickham | Beginner-Int | [Yes](https://r4ds.hadley.nz/) | Best DS book for R users |

## Reading Order Recommendations

### For Complete Beginners (Start Here)
1. Think Stats (free) → Python Data Science Handbook (free) → Hands-On ML (3rd ed.)

### For DS Career Track
1. Python Data Science Handbook → Hands-On ML → ISLR → Statistical Rethinking

### For MLE Career Track
1. Hands-On ML → Designing ML Systems → Designing Data-Intensive Applications → D2L

### For AI Engineer Career Track
1. Build an LLM from Scratch → NLP with Transformers → AI Engineering (Chip Huyen) → Prompt Engineering for LLMs
