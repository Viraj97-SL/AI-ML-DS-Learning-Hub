## Recommended Learning Order (By Track)

### Data Scientist
1. Python for Everybody (if new to Python)
2. Statistics and Probability — Khan Academy
3. Machine Learning Specialization — Andrew Ng
4. Practical Deep Learning for Coders — fast.ai
5. Kaggle competitions (ongoing practice)

### ML Engineer
1. Machine Learning Specialization — Andrew Ng
2. Full Stack Deep Learning — fullstackdeeplearning.com
3. MLOps Zoomcamp — DataTalks.Club
4. Practical Data Science on AWS or GCP equivalent

### AI Engineer
1. Neural Networks: Zero to Hero — Andrej Karpathy
2. DeepLearning.AI Short Courses (start with Prompt Engineering)
3. LangChain Academy
4. Hugging Face NLP Course
5. Build projects throughout
