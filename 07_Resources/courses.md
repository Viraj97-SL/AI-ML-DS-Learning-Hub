## 1. Free Learning Platforms

### 🆓 Kaggle Learn
> **Best for:** Practical ML/DS skills with instant runnable notebooks
- **URL:** https://www.kaggle.com/learn
- **Highlights:** Python, Pandas, ML Intro, Intermediate ML, Feature Engineering, NLP, Computer Vision, SQL, Geospatial Analysis, Time Series, Game AI
- **Strength:** Micro-courses (4–8 hours each) with Kaggle kernels — run code directly in browser, no setup
- **Certificate:** Yes, free and shareable

### 🆓 fast.ai
> **Best for:** Practitioners who want deep learning without heavy math prerequisites first
- **URL:** https://www.fast.ai
- **Courses:**
  - *Practical Deep Learning for Coders* — CNNs, tabular, NLP, diffusion models (PyTorch)
  - *Practical Deep Learning Part 2* — Foundations from scratch (stable diffusion internals)
  - *Computational Linear Algebra* — Numerical methods for ML
- **Strength:** Top-down approach (applications before theory), exceptional teaching quality, Jeremy Howard is a legend
- **Free Book:** "Deep Learning for Coders with fastai and PyTorch" at https://book.fast.ai

### 🆓 Google Machine Learning Crash Course
> **Best for:** Beginner-friendly ML concepts with interactive visualizations
- **URL:** https://developers.google.com/machine-learning/crash-course
- **Highlights:** 25+ lessons, 40+ exercises, TensorFlow playground, real-world case studies
- **Additional Google ML education:** https://developers.google.com/machine-learning

### 🆓 Microsoft Learn
> **Best for:** Azure ecosystem, AI/ML fundamentals, official cert prep
- **URL:** https://learn.microsoft.com/en-us/training/
- **Key Learning Paths:**
  - AI-900: Azure AI Fundamentals
  - DP-900: Azure Data Fundamentals
  - AI-102: Azure AI Engineer (full prep, free)
  - DP-100: Azure Data Scientist (full prep, free)
- **Strength:** Free sandbox environments, hands-on labs with no local install needed

### 🆓 IBM Skills Network / Cognitive Class
> **Best for:** DS/ML fundamentals with IBM Cloud integration
- **URL:** https://skills.network and https://cognitiveclass.ai
- **Notable Courses:**
  - Machine Learning with Python
  - Deep Learning Fundamentals
  - Apache Spark for Beginners
  - Data Science Methodology
- **Certificate:** Free digital badges via Credly (recognized by employers)

### 🆓 Harvard CS50 Series
> **Best for:** World-class CS fundamentals — the best free courses on Earth
- **URL:** https://cs50.harvard.edu
- **Cost:** Free to take; verified certificate ~$149 (optional)
- **Key Courses:**

| Course | Description | Link |
|--------|-------------|------|
| **CS50x** | Introduction to Computer Science — best beginner CS course ever made | https://cs50.harvard.edu/x |
| **CS50 AI** | Intro to AI with Python — search, ML, neural nets, NLP | https://cs50.harvard.edu/ai |
| **CS50 Python** | Introduction to Python programming | https://cs50.harvard.edu/python |
| **CS50 SQL** | Introduction to databases and SQL | https://cs50.harvard.edu/sql |
| **CS50 Web** | Web programming with Python and JavaScript | https://cs50.harvard.edu/web |

### 🎓 DataCamp
> **Best for:** Structured, guided DS/ML career tracks with interactive browser coding
- **URL:** https://www.datacamp.com
- **Cost:** First chapter of every course is free; $25/month or ~$149/year (sale price)
- **Career Tracks:** Data Scientist, ML Scientist, Data Engineer, AI Engineer, Data Analyst
- **Strength:** Progressive career tracks, real datasets, built-in coding environment, skill assessments

### 🎓 Coursera & edX (Audit Mode)
> **Best for:** University-level courses — free to learn, pay only if you want the certificate
- **URLs:** https://www.coursera.org | https://www.edx.org
- **How to audit on Coursera:** Enroll → click "Audit this course" → access all videos & readings (graded assignments locked)
- **How to audit on edX:** Enroll → select "Audit" track
- **Top auditable courses:**
  - MIT 6.S191: Introduction to Deep Learning (https://introtodeeplearning.com — free directly too)
  - Stanford CS229: ML (materials at https://cs229.stanford.edu, lectures on YouTube)
  - Stanford CS224N: NLP with Deep Learning (YouTube)

### 🆓 Codecademy
> **Best for:** Interactive beginner coding in Python, SQL, data analysis
- **URL:** https://www.codecademy.com
- **Key courses (free tier):** Python 3, SQL, Learn HTML/CSS
- **Pro ($20/month):** Data Science path, Machine Learning, Data Engineering

### 🆓 Data 365
> **Best for:** Data analytics, business intelligence, beginner-friendly DS
- **URL:** https://365datascience.com
- **Highlights:** Python, SQL, Tableau, statistics, ML — structured beginner-to-job path

### 🆓 Hugging Face Learn
> **Best for:** Transformers, NLP, diffusion models — the official source
- **URL:** https://huggingface.co/learn
- **Courses:**
  - NLP Course (transformers, fine-tuning, datasets)
  - Deep RL Course (reinforcement learning with RL libraries)
  - Audio Processing Course
  - Diffusion Models Course
- **Strength:** Free, hands-on, Colab notebooks, huge community

### 🆓 MIT OpenCourseWare (OCW)
> **Best for:** Rigorous mathematical foundations — grad-level material for free
- **URL:** https://ocw.mit.edu
- **Essential courses:**
  - 18.06: Linear Algebra (Prof. Gilbert Strang) — legendary
  - 18.065: Matrix Methods in Data Analysis and ML
  - 6.0002: Intro to Computational Thinking and Data Science
  - 6.034: Artificial Intelligence

## 8. Recommended Learning Paths

### Path A: Data Scientist (12–18 months)
```
Month 1–2:   CS50x (Harvard, free) OR Kaggle Learn Python + Pandas
Month 3–4:   Statistics (Khan Academy + StatQuest YouTube)
Month 5–6:   Machine Learning Specialization (Andrew Ng, Coursera)
Month 7–8:   Deep Learning Specialization (DeepLearning.AI)
Month 9–10:  Kaggle competitions + this repo's DS notebooks
Month 11–12: Cloud cert (AWS AI Practitioner or GCP Associate Cloud Engineer)
Month 13+:   Specialize based on interest (NLP → CS224N, CV → CS231N, etc.)
```

### Path B: ML Engineer (12–18 months)
```
Month 1–2:   Python SWE (CS50 Python + OOP, testing, packaging)
Month 3–4:   Docker + FastAPI + SQL (Udemy Jose Portilla courses)
Month 5–6:   ML fundamentals (Kaggle Learn full ML track)
Month 7–8:   MLOps (Full Stack Deep Learning + MLOps Zoomcamp)
Month 9–10:  Cloud cert (AWS ML Specialty or GCP Professional ML Engineer)
Month 11–12: Databricks Spark Associate cert + this repo's MLE notebooks
Month 13+:   Kubernetes, distributed training, platform engineering
```

### Path C: AI Engineer (12–18 months)
```
Month 1–2:   Python + CS50 AI (Harvard, free)
Month 3–4:   LLM fundamentals (fast.ai + Andrej Karpathy "Zero to Hero" YouTube)
Month 5–6:   API skills (DeepLearning.AI short courses — all free)
Month 7–8:   LangChain + RAG systems (Hugging Face NLP Course + LangChain Academy)
Month 9–10:  AI agents + evals + this repo's AIE notebooks
Month 11–12: Cloud AI cert (Azure AI-102 or AWS ML Specialty)
Month 13+:   Fine-tuning, multi-agent systems, production AI reliability
```

### Path D: Career Switcher — Fast Track (6 months intensive)
```
Week 1–4:    CS50x + Python intensive (2–3 hours/day)
Week 5–8:    Kaggle Learn: Python, Pandas, ML, SQL (all free)
Week 9–12:   Andrew Ng ML Specialization (Coursera, audit free)
Week 13–16:  Hands-on projects (this repo's labs, Kaggle competitions)
Week 17–20:  Interview prep (this repo's 06_Interview_Prep section)
Week 21–24:  Entry-level cert (AWS AI Practitioner $100 or Azure AI-900 $165)
```
