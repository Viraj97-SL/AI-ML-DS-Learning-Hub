## What to Learn by Role

### Data Scientist (2025–2026 Priority)
1. **Hybrid search** and re-ranking — almost every DS role now touches RAG pipelines
2. **LLM evaluation** — designing task-specific evals beyond benchmark scores
3. **Synthetic data generation** — for fine-tuning and augmentation
4. **AI governance / EU AI Act** — compliance is now a DS responsibility in EU companies

### ML Engineer (2025–2026 Priority)
1. **DPO / GRPO fine-tuning** — replacing RLHF in most production workflows
2. **LLMOps** — LangSmith, Arize Phoenix, Helicone for production monitoring
3. **Small language models** — cost-optimized inference with Phi-4, Gemma 2
4. **Vector database operations** — MongoDB Atlas Vector Search, schema design for RAG

### AI Engineer (2025–2026 Priority)
1. **Agentic frameworks** — LangGraph, AutoGen v0.4, Amazon Strands
2. **MCP server development** — building and hosting MCP servers is the new "build an API"
3. **A2A protocol** — cross-agent communication patterns
4. **Agent reliability patterns** — checkpointing, compensation, VIGIL-style supervision
5. **ReasoningBank / episodic memory** — making agents that learn across sessions
