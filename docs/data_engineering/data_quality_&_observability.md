### Data Engineering

#### Overview

Data Engineering is the backbone of the modern data stack. It focuses on designing, building, and maintaining the systems and infrastructure that allow for the large-scale collection, storage, processing, and analysis of data. While data scientists analyze data to extract insights, data engineers build the reliable pipelines and platforms that make this analysis possible. A solid understanding of data engineering principles is crucial for anyone looking to work with data at scale, as it ensures that data is accessible, clean, and ready for analysis or machine learning models.

#### Projects/Exercises

1.  **Guided Project: FMCG Data Pipeline**
    *   Follow the [End to End Data Engineering Project using Databricks](https://www.youtube.com/watch?v=U6ZUKWdfSLY) tutorial. This will guide you through building a complete pipeline, giving you hands-on experience with a popular industry tool.

2.  **Build a Personal Data Pipeline**
    *   Choose a public API that interests you (e.g., weather, stock prices, public transit).
    *   Write a script to extract data from the API on a schedule (e.g., once a day).
    *   Perform basic cleaning and transformation on the data.
    *   Load the cleaned data into a database or cloud data warehouse of your choice. This is a great opportunity to apply what you learned in the Snowflake crash course.

3.  **Improve an Existing Dataset**
    *   Find a messy dataset from a public source.
    *   Apply the techniques from the [Data Cleaning/Data Preprocessing Guide](https://www.youtube.com/watch?v=GP-2634exqA) to clean, normalize, and prepare the data for analysis. Document your steps and explain why each transformation was necessary.
