### Generative AI & Foundation Models

#### Overview

Generative AI is a branch of artificial intelligence focused on creating new, original content—such as text, images, music, and code—that resembles data it was trained on. The engine behind many of these powerful tools is a **Foundation Model**, which is a large-scale model trained on a vast quantity of broad data. These models (like GPT for text or Stable Diffusion for images) can then be adapted for a wide range of specific tasks. Understanding how to effectively use and prompt these models is a critical skill in modern data science and software development.

#### Projects & Exercises

1.  **Creative Story Generator**: Use a free text-generation model (like a public version of GPT or a model on Hugging Face) to co-write a short story. Start with an opening sentence and experiment with different prompts to guide the narrative, tone, and character development.
2.  **Image Prompting Challenge**: Use a text-to-image generator to create a specific scene. Start with a simple prompt like "a red car" and iteratively add details about the style (e.g., "in the style of a watercolor painting"), setting ("on a futuristic city street at night"), and composition. Note how small changes in your prompt affect the final image.
3.  **Text Summarizer**: Find a long article online and use a large language model to summarize it. Ask for different types of summaries: a single sentence, a three-bullet-point list, and a paragraph for a non-expert audience. This helps you practice controlling the model's output format.
