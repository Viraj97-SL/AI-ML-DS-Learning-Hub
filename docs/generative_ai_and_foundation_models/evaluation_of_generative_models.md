### Generative AI & Foundation Models

#### Overview

Generative AI is a branch of artificial intelligence focused on creating new, original content—such as text, images, code, or music—by learning patterns from existing data. At the core of this revolution are **Foundation Models**, which are large-scale models like GPT (Generative Pre-trained Transformer) or Stable Diffusion, trained on vast amounts of data. These models serve as a powerful base that can be adapted or fine-tuned for a wide range of specific tasks, from writing emails to generating photorealistic art. Understanding these concepts is key to leveraging the most advanced capabilities in AI today.

#### Projects & Exercises

*   **Text Generation with a Pre-trained Model**: Use a library like Hugging Face's `transformers` to load a pre-trained language model (e.g., GPT-2). Write a simple script to generate text based on a starting prompt. Experiment with different prompts and model parameters to see how the output changes.
*   **Image Generation with Prompts**: Use a free image generation tool or API (like those available through Hugging Face Spaces or other services). Practice "prompt engineering" by writing detailed text descriptions to create specific images. Try to generate an image of a "blue orange" or a "cyberpunk-style cat" and refine your prompts to improve the results.
*   **Build a Simple RAG Chatbot**: RAG (Retrieval-Augmented Generation) is a powerful technique where a model retrieves information from a knowledge base before generating an answer. Create a simple version: write a Python script that takes a user's question, finds the most relevant paragraph from a text file you provide, and then feeds that paragraph along with the original question into a language model to generate a final answer.
