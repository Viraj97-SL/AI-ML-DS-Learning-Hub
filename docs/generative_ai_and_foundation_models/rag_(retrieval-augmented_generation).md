
## Generative AI & Foundation Models

### Overview

Generative AI is a branch of artificial intelligence focused on creating new, original content—from text and images to music and code. The engine behind many of these powerful tools is a **Foundation Model**. These are large-scale models, like GPT or Stable Diffusion, that are pre-trained on vast amounts of data. This pre-training gives them a broad understanding of language, patterns, and concepts, allowing them to be adapted (or "fine-tuned") for a wide range of specific generative tasks. This section explores the core principles of these models and how they are used to build modern AI applications.

### Projects & Exercises

1.  **Text Generation with a Pre-trained Model**: Use a library like Hugging Face's `transformers` to load a pre-trained language model (e.g., GPT-2). Write a simple script that takes a starting phrase (a "prompt") and generates a paragraph of text.
2.  **Explore Prompt Engineering**: Choose a publicly available text-to-image or text-to-text model. Craft five different prompts to generate the same core subject (e.g., "a dog on a skateboard"). Vary the style, details, and constraints in your prompts to see how it changes the output. Document your findings.
3.  **Summarization Task**: Find a long article online. Use a foundation model specialized for summarization (like T5 or BART) to create a concise, one-paragraph summary. Compare the model's output to the original article's key points.
