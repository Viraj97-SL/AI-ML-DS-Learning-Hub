
# Python for Data Science

### Overview

Python is the dominant programming language for data science due to its simplicity, versatility, and the powerful ecosystem of open-source libraries it supports. For data science, Python is used for everything from cleaning and analyzing data to building and deploying complex machine learning models. Key libraries like **Pandas** provide high-performance data structures for data manipulation, **NumPy** offers essential support for numerical computing, **Matplotlib** and **Seaborn** are used for data visualization, and **Scikit-learn** provides a comprehensive toolkit for machine learning. This section provides resources to help you get started using Python for data analysis and machine learning tasks.

### Projects & Exercises

1.  **Exploratory Data Analysis (EDA):**
    *   Find a simple dataset online (e.g., the Titanic dataset from Kaggle or the Iris dataset included with Scikit-learn).
    *   Use **Pandas** to load the data into a DataFrame.
    *   Practice data cleaning: check for and handle missing values.
    *   Calculate summary statistics (mean, median, standard deviation) for numerical columns.
    *   Use `value_counts()` to explore categorical columns.

2.  **Data Visualization:**
    *   Using the same dataset from the EDA project, create visualizations with **Matplotlib** or **Seaborn**.
    *   Create a histogram to view the distribution of a numerical feature (e.g., age).
    *   Create a bar chart to show the count of different categories (e.g., passenger class).
    *   Create a scatter plot to explore the relationship between two numerical features.

3.  **Build a Simple Predictive Model:**
    *   Using a clean dataset (like the Iris or Boston Housing dataset), use **Scikit-learn** to build a basic machine learning model.
    *   Split your data into training and testing sets.
    *   Train a simple model, such as a `LogisticRegression` for classification or `LinearRegression` for regression.
    *   Evaluate your model's performance on the test set.
